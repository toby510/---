### 本章核心知识点总结
本章是讲解如何加快训练，即word2vec的高速化
#### 1-输入one-hot和W_in的乘积，通过W[idx]来快速获取每个单词词向量
#### 2-多分类转化为二分类思想
输入是n个单词，预测结果是n个单词的概率输出，转化为：输入是上下文，目前是target时，结果是对还是错
在计算h*W_out时，W_out取idx列向量即可
#### 3-负采样思想
1. 正样本标签为1，负样本标签为0
2. 负样本如何采集才公平，公式：p=单词的0.75次方，即高频词概率要降低，地频词概率要升高
3. 负样本不一定都要训练，只是根据以上规则采集部分负样本后，进行正向和反向训练即可
